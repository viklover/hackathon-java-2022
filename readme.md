User Guide
---
Наш проект имеет backend и frontend.

Backend написан на Java c использованием Maven. \
Frontend работает как html-страница с ванильным JS.

Стороны взаимодействует друг с другом через HttpServer, который работает на серверной стороне.

Более подробное описание серверной части и инструкцию по устновке вы можете посмотреть по ссылке ниже
